Getting Started

How to install a Wordpress theme

To start the theme installation first locate "quantum.zip" within the package you already downloaded from Themeforest.Unpack that file to "Desktop" where you can find it easily. Than go to "Appearance" - "Themes" - "Install Themes" - "Upload" and point the file-dialog at "quantum.zip"

Creating Home Masonry

 

Quantum includes a lot of page templates which can be used as homepages or for some other page. For example we have Home masonry page which is equally good for either your homepage or portfolio page

The page template name for this page is "Home Masonry", to create it please use the instructions below

After you choose the page template, options for that particular page will show up.

In the same exact way you create all other page templates, which I will explain below

Creating clients page

To create a Clients page, you need to create a new Page first by going to Pages - Add New and choose "Clients" as a page template from the drop-down on the right.

Immediately there should appear new meta-boxes.

Page Heading : Choose the page heading title

Clients : From this field you can add as many clients as you want, by clicking add client

Client Logo : Upload the client Logo from this field.

Client URL : Enter the client web-site

 

Creating contact page

To create a Contact page, you need to create a new Page first by going to Pages - Add New and choose "Contact" as a page template from the drop-down on the right.

Immediately there should appear new meta-boxes.

You have two options here, to use a google map, or a world map to show where your company is located in a more minimal way.

If you choose google map:

Map : Click on the map where your location is.

Map Zoom : Choose the zoom level

If you choose Custom map:

You can add as many location icons by double clicking on the map. By dragging you can change their position and by clicking on them you set the city name.

E-mail : Type the e-mail address where you want to accept contact form messages.

Success Message : Success message which will appear when message is sent through contact form

Adress : Type a contact note, or your address

 

 

 

Creating services page

To create a Services page, you need to create a new Page first by going to Pages - Add New and choose "Services" as a page template from the drop-down on the right.

Immediately there should appear new meta-boxes.

Services : Add as many services as you need from this field. Click add service to add a new one.Once you click add row, fill the individual fields for each service.

Creating team page

To create a Team page, you need to create a new Page first by going to Pages - Add New and choose "Team" as a page template from the drop-down on the right.

Immediately there should appear new meta-boxes.

Team Members : Add as many team members as you need from this field.Once you click add row, fill the individual fields for each service.

Adding projects, using project page alternative designs

To create a new project, you need to go to Portfolio - Add new Post.

As you may have seen in the live demo, there are three alternative designs for the project page:

1. Project page with simple slider with description of the project

2. Case study style project page

3. Full screen, flickr inspired gallery style (only images)

Adding content is very easy in each three. First you select one of the project page styles, then upload your project images in the "Slider" section, if you want to use an video instead of an image you paste the link in the box next to image.

How to set homepage and blog

To set a page as your homepage you have to go to Settings > Reading and then choose the option " A static page (select below)" For the "Front page" select the page you want to be your homepage For the "Posts page" select the blog page
